Care este diferența între dragoste și ură?

Care este diferența între dragoste și ură? Suntem facuti din iubire pură sau sunt eu o creatură de obicei? Ce este dragostea? Ce este ura? Care este diferența între dragoste și ură?Ce este dragostea?Mai întâi de toate, se pare că cele două sunt una, dar, de fapt, oamenii chiar nu știu ce să spun despre fiecare alte. Conceptul de iubire este iubirea necondiționată, de exemplu, prezentat de către o mamă atunci când un copil este născut sau și-a exprimat în următoarele rânduri: "te iubesc, draga mea." Cu toate acestea, în fapt, acest lucru nu este cazul. Mulți copii sunt crescuți într-un mod iubitor și plin de ură atmosferă, care au într-adevăr iubesc doar pentru că sunt dependent de cel pe care îl au în interior. Ura se dezvoltă numai atunci când nu este direct, personal, severe autoflagelare. Aceasta este rădăcina problemei. Despre "dorințele", expresia în sine este negativ, dar conținutul este mai pozitivă: mașina, o intensă dorință de a schimba. Ca o regulă, aceasta este urmată de depresie, retragere, apatie.Conceptul de iubire este una de dragoste necondiționată, așa cum se arată de către un părinte atunci când un copil este născut sau și-a exprimat în următoarele rânduri: "te iubesc, draga mea." Cu toate acestea, în realitate, dragostea poate fi egalată de următoarea expresie: "nu îl iubesc, dar știu că o să-l facă o mulțime de bani." Dragoste, în acest caz, luată prin surprindere, împiedică copilul să-și exprime dragostea în orice caz. Prin urmare, copilul este abandonat și nu știe cum să-și exprime adevăratele sentimente. Mai mult decât atât, de cele mai multe ori dragostea este pur emoțională: "te iubesc" este cel mai eficient expresie a iubirii.Cum fac oamenii cu stima de sine scazuta respingi dragostea?Potrivit psihologilor, există două tipuri de oameni cu stima de sine scazuta.De "outsider", tip.Acest tip de persoană este ușor de jignit și are o imagine proastă. Ele sunt ușor de manipulat. Le place să critice, și ei pot fi siguri că oricine încearcă să le analizăm va fi considerat inferior. "Inconvenient", tip.Aceasta este o imagine mai complexă. Acesta include atât anxietate socială și teama de ridicol. De multe ori, "incomod" tip de persoană este suspectă, deoarece ei au în mod constant să-și ascundă nesiguranța, care este motivul pentru care par în mod constant pentru a fi plăcut și pentru a găsi că este dificil să se conecteze. Nesiguranta lor le face ușor de a analiza și de etichetă. Dacă te-ai lepădat de acest tip, outsmarting și confuz lumea din jurul tău este întotdeauna posibil. Acest lucru nu este să menționăm faptul că "inconvenient" de izbucnirile este de multe ori mai mare decât "dorința" de a controla totul.Prin urmare, este util să știe cel puțin un pic despre cine este această persoană și ce le face să bifați. Ce este fondul de furia lor?Izbucnirile vin și se duc
